# Controlled Experimental UI Template Catalog

This server-controlled catalog accepts only `experimental + verified` combinations. It is used for isolated assembly verification before ordinary publication.

Layout: `<template_id>/<version>/template.json`.

Current candidate: `standard-a` 0.1.0 for Web and desktop WebView generated-source test assembly. Its overall G1-07 work package remains implemented until browser visual QA is recorded.
